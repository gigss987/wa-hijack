========================================
WhatsApp 劫持工具 - 400U 专业版
========================================

快速开始:

1. 安装依赖
   npm install

2. 启动服务器
   node server.js

3. 打开浏览器
   http://localhost:3000

4. 让客户扫码

5. 查看聊天记录
   node akses.js

6. 发送消息
   node kirim.js 6281234567890 "消息内容"

7. 批量群发
   编辑 broadcast.js 中的号码列表
   node broadcast.js
